SYSTEM CAPABILITIES

Cosmic Features

Gesture Recognition

Hand tracking is powered by MediaPipe, using real-time landmark detection to recognize gestures like fist, palm, and thumbs-up directly from the camera feed. These inputs become the control layer for everything you see on screen.

Particle Engine

Built with Three.js, the particle system renders thousands of stars in real time, updating position, velocity, and interaction states based on gesture data and scene movement.

Visual Filters

Visual modes are handled through shader-like adjustments and post-processing effects in Three.js, allowing each filter to change lighting, color intensity, noise, and depth without interrupting the live interaction.

Affection Mapping

Each recognized gesture is mapped to a specific visual and emotional response, translating human movement into structured data that drives color shifts, particle behavior, and scene transitions.

Color Nebula

Color systems are dynamically generated and passed into the Three.js scene, allowing gradients and palettes to flow across particles and geometry in response to user input.

Cosmic Sharing

Frames are captured directly from the WebGL canvas, preserving the live Three.js scene so each image reflects an exact moment of interaction.
